競賽常見演算法

基礎套路

  • 雙指針 / 滑動窗口、前綴和 / 差分
  • 二分查找（答案二分、邊界判定）
  • 排序家族（快排、歸併、桶/基數）
  • 貪心（區間選擇、排程、字典序）
O(n log n)單調性不變量

圖論

  • BFS/DFS、拓撲排序、連通分量
  • 最短路：Dijkstra、SPFA、Floyd
  • 最小生成樹：Kruskal / Prim（並查集）
  • 樹：LCA、樹徑、DFS 次序、樹上 DP
鄰接表堆優先隊列並查集

資料結構

  • 堆、單調棧/隊列
  • 樹狀數組、線段樹（含懶標記）
  • 字典樹（Trie）、並查集（DSU）
  • 哈希（開放定址 / 滾動哈希）
區間查詢動態維護位運算

數學 & DP

  • 數論：GCD、快速冪、篩法、模運算
  • 組合數 & 卡特蘭、容斥、概率期望
  • DP：背包、區間、樹形、狀壓、數位 DP
  • 字符串：KMP、Z 函數、Manacher
轉移設計狀態壓縮複雜度
